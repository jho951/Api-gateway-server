### 역할
- 인증 결과를 신뢰 가능한 방식으로 연결하는 계층입니다.
- 인증을 직접 소유하면 안됩니다.


### 책임

- 단일 진입점
- 라우팅
- 인증 여부 판별
- 토큰/세션 검증 연계
- 내부 서비스 전달용 사용자 컨텍스트 주입
- CORS / 보안 헤더 / 공통 필터 처리
- 클라이언트 종류별 진입 정책 분기
- 로깅 / 추적 ID 전달

--- 

## 비책임
- 사용자 비즈니스 로직 처리
- 권한 정책의 상세 판단
- 사용자 정보 DB 조회의 주체
- 에디터 도메인 처리
- 인증서버 역할 대체
- 토큰 발급 주체 역할
